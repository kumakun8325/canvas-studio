# /learn - セッション学習の永続化

このセッションで発見した知見を `.claude/memory/learned.md` に記録してください。

## 記録すべき内容

### 1. デバッグテクニック
- 問題の特定に役立った手法
- エラーの根本原因と解決策

### 2. ワークアラウンド
- ライブラリの制限を回避した方法
- 一時的な対処法とその理由

### 3. プロジェクト固有のパターン
- このコードベースで発見した暗黙のルール
- 既存コードから学んだベストプラクティス

### 4. 失敗と学び
- うまくいかなかったアプローチ
- 次回避けるべきパターン

## フォーマット

```markdown
## [日付] セッション学習

### 発見事項
- [カテゴリ] 内容

### 今後の推奨事項
- 具体的なアクション
```

## 手順

1. このセッションでの主要な学びを振り返る
2. 上記カテゴリに分類
3. `.claude/memory/learned.md` に追記（既存内容は保持）
4. 重要な発見は `.claude/rules/` へのルール追加を検討
